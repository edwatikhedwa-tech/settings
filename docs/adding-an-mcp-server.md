# Adding an MCP Server

1. Найди официальный источник сервера.
2. Проверь владельца, пакет, transport, permissions и способ health check.
3. Добавь запись в `mcp/servers.yaml`.
4. Не записывай значения секретов, только имена.
5. Проверь `mcp/schema.json`.
6. Добавь документацию по отключению и rollback.
7. Проверь, что сервер не обходит авторизацию, условия сервиса, rate limits или приватность третьих лиц.
