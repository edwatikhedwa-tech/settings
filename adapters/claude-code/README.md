# Claude Code Adapter

Подтверждено официальной документацией Anthropic/Claude Code:

- `claude mcp` управляет MCP-серверами.
- `--mcp-config` и `--strict-mcp-config` позволяют использовать явный MCP config для запуска.
- Settings хранятся в JSON и могут включать MCP-related параметры.

Этот репозиторий не переносит Codex TOML напрямую в Claude Code. Инструкции генерируются отдельно в `generated/claude-code/instructions.md`.
