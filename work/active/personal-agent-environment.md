# Личное окружение AI-агентов для разработки и исследований

- Status: completed
- Started: 2026-07-19
- Last updated: 2026-07-19

## Goal

Убрать повторную настройку и ручную рутину при работе с AI-агентами: один запрос пользователя должен запускать поиск, извлечение доступного содержания, проверку значимых утверждений и сохранение проверяемого результата.

## Current State

Исследование завершено: 58 уникальных результатов, 49 кандидатов, шесть расшифровок через Supadata. Полный текст роликов хранится только в ответе активного запуска и не коммитится. Отчёт обновлён в `knowledge/youtube/personal-agent-environment.md`.

## Decisions

- Использовать API-first извлечение текста через локальный `youtube_transcript_extract`; браузерный захват оставить запасным вариантом.
- Разделять глобальные предпочтения, правила проекта, skills, MCP и карточки работы вместо одного большого универсального промпта.
- Перед автоматизацией описывать и вручную проверять процесс; затем добавлять ограниченную, измеримую автоматизацию.
- Не применять кандидаты на изменение `AGENTS.md`, MCP и конфигурации без отдельного подтверждения.
- Фиксировать стоимость и бесплатные лимиты в каждом исследовании; Supadata имеет лимит 100 запросов в месяц, в этом запуске израсходовано 11.
