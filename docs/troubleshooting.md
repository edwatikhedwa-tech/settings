# Troubleshooting

## Git не найден

Технически: команда `git` не найдена в PATH.

Простыми словами: Windows не знает, где установлен Git.

Безопасное действие: установить Git с официального сайта, затем открыть новый терминал.

## GitHub CLI не найден

Технически: команда `gh` не найдена в PATH.

Простыми словами: скрипт не сможет сам создать приватный GitHub-репозиторий через CLI.

Безопасное действие: установить GitHub CLI и авторизоваться официальной командой `gh auth login`.

## Codex или Claude Code не найден

Клиент будет пропущен. Это не ошибка установки всего репозитория.
