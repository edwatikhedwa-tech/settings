# ChatGPT Adapter

Подтверждено официальными материалами OpenAI:

- Custom Instructions применяются к чатам пользователя.
- Projects могут содержать файлы и инструкции проекта.
- Apps/connectors подключаются пользователем через ChatGPT и OAuth.

Не найдено подтверждение, что обычный новый чат ChatGPT автоматически загружает правила из GitHub-репозитория. Поэтому этот адаптер создаёт ручной артефакт `generated/chatgpt/instructions.md`.
