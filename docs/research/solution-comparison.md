# Solution Comparison

Дата доступа: 2026-07-18.

## Исследованные подходы

| Подход | Преимущества | Недостатки | Риски | Итог |
| --- | --- | --- | --- | --- |
| Dotfiles-репозиторий | Хорошо ложится на GitHub, легко версионировать, удобно для `AGENTS.md` и конфигов | На Windows разные клиенты используют разные пути и форматы | Можно случайно перезаписать пользовательские файлы | Использовать как внешний контейнер, но не копировать файлы напрямую без адаптеров |
| Идемпотентный bootstrap | Управляет dry-run, backup, merge и диагностикой | Требует больше кода и тестов | Ошибка скрипта может затронуть настройки | Выбран как основной механизм применения |
| Шаблоны и генераторы | Один канонический источник, меньше ручного дублирования | Нужна проверка, что generated-файлы актуальны | Неправильная генерация может разойтись с исходниками | Выбран для клиентских инструкций |
| Официальные механизмы клиентов | Самый безопасный путь, когда документирован | Механизмы отличаются между Codex, Claude Code и ChatGPT | Нельзя переносить формат одного клиента в другой | Использовать через отдельные адаптеры |

## Почему выбран итоговый подход

Итоговая архитектура сочетает dotfiles-подход как GitHub-источник истины, идемпотентный PowerShell-bootstrap как безопасный установщик и генератор клиентских артефактов как защиту от ручного дублирования.

## Подтверждённые факты

- Codex документирует `AGENTS.md`, пользовательский и проектный `config.toml`, hooks и MCP.
- Claude Code документирует `claude mcp`, настройки JSON и флаги для MCP-конфигурации.
- MCP спецификация документирует `stdio` и Streamable HTTP.
- ChatGPT документирует Custom Instructions, Projects и Apps, но не подтверждает автоматическую синхронизацию обычного чата с GitHub-репозиторием.

## Архитектурные решения

- `policies/` является каноническим источником правил.
- `scripts/build-configs.py` генерирует клиентские инструкции.
- `mcp/servers.yaml` является клиент-независимым реестром MCP.
- Установка по умолчанию работает в режиме плана; применение требует `-Apply`.

## Ограничения

- В текущей среде не подтверждена работа `git`, `gh`, Codex CLI и Claude Code CLI.
- Автоматическая настройка ChatGPT ограничена ручной вставкой или официальными apps/projects/custom GPT механизмами.
