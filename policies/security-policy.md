# Security Policy

В репозиторий запрещено сохранять:

- API-ключи;
- OAuth-токены;
- пароли;
- cookies;
- приватные SSH-ключи;
- содержимое `.env`;
- пользовательские сессии;
- секретные заголовки авторизации.

MCP-сервер добавляется только после проверки официального источника, владельца, пакета, способа запуска, разрешений и рисков выполнения кода.

Скрипты не должны запускать удалённый код через `Invoke-Expression` без отдельного обоснования и явного подтверждения.
