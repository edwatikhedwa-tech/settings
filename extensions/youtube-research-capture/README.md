# YouTube Research Capture

Бесплатное локальное расширение для автоматического получения в `topic-research` текста, который уже отображается в YouTube Summary или открыт в YouTube.

## Что делает

- Автоматически обрабатывает очередь до шести роликов, которую `topic-research` создал по введённой теме.
- Берёт текст из уже отображённого виджета YouTube Summary, в том числе когда публичная панель субтитров YouTube недоступна.
- Может сохранить текст из уже открытой панели расшифровки YouTube.
- Передаёт данные только на `http://127.0.0.1:8765`.
- Сохраняет исходный текст в `private/research-cache/youtube-captures/`, который исключён из Git.

## Однократная установка

1. В Chrome откройте `chrome://extensions`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked` и выберите эту папку: `extensions/youtube-research-capture`.

## Использование

1. Приёмник запускается командой `scripts/start-youtube-capture.ps1 -Apply`.
2. В Codex запустите `topic-research`. Он сам создаёт очередь из шести роликов, а расширение открывает их в фоне и получает текст из виджета YouTube Summary.
3. Навык ждёт статусы очереди, использует доступные тексты и отмечает недоступные ролики без попыток обхода ограничений.

Кнопки захвата из буфера и видимой расшифровки в popup остаются только запасным способом диагностики, а не частью обычного исследования.

Расширение не получает cookies, пароли, токены, закрытые субтитры, список вкладок или данные других сайтов. Оно не обходит CAPTCHA, авторизацию, лимиты или платные функции YouTube Summary.
