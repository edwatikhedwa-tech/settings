# YouTube Research

Цель навыка - собрать проверяемые практические выводы, а не просто список роликов.

1. Уточни тему, язык, период, каналы и критерий полезности, если они заметно меняют результат.
2. Используй только разрешённый YouTube-инструмент, официальный API или подключённый MCP. Не обходи ограничения платформы и не извлекай недоступные субтитры.
3. Собери ограниченную, объяснимую выборку: запросы, каналы или плейлисты должны быть записаны в заметке.
4. Для каждого важного ролика сохрани URL, автора, дату, релевантный фрагмент, статус субтитров и краткий вывод.
5. Сверь сильные утверждения с первичными источниками. Мнение автора видео помечай как мнение.
6. Сохрани результат в `knowledge/youtube/<topic>.md` по шаблону. Заверши разделами `What changed`, `Open questions` и `Next action`.

Нельзя обещать анализ всех роликов YouTube: выборка всегда ограничена поисковым запросом, каналами, периодом, доступом API и доступностью контента.
