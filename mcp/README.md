# MCP Registry

`servers.yaml` является клиент-независимым реестром. Он не является готовым конфигом Codex, Claude Code или ChatGPT.

Адаптеры должны преобразовывать только подтверждённые поля в формат конкретного клиента. Секреты не хранятся в этом файле; здесь разрешены только имена переменных или секретов.
