# Task Memory

Для длительной или повторяющейся задачи используй одну карточку в `work/active/`.

Для новой многошаговой задачи используй `templates/task-brief.md` как краткую постановку. Не требуй шаблон для простого ответа или одного обратимого действия.

Перед существенным изменением прочитай её разделы `Current State`, `Decisions` и `Next Action`.
После существенного этапа обнови кратко:

- фактический результат;
- принятое решение и его причину;
- ссылки на важные источники и артефакты;
- один конкретный следующий шаг.

Не записывай токены, пароли, cookies, приватные данные или полные стенограммы без необходимости. Завершённые карточки перемещай в `work/done/` только после явного подтверждения результата.
