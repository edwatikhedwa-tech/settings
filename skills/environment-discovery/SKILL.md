# Environment Discovery

Проверь ОС, PowerShell, наличие Git, GitHub CLI, Codex, Claude Code, Node.js, Python и доступность рабочих папок.

Не изменяй файлы во время discovery. Результат разделяй на `[OK]`, `[WARN]`, `[ERROR]` и `[SKIP]`.
