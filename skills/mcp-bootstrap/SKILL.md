# MCP Bootstrap

Перед добавлением MCP проверь официальный источник, владельца, пакет, транспорт, разрешения, секреты и health check.

Секреты храни только вне Git. В `mcp/servers.yaml` разрешены только имена секретов.

Не добавляй MCP-сервер, если его основная цель — обход авторизации, CAPTCHA, rate limits, платных ограничений, условий сервиса или приватности.
