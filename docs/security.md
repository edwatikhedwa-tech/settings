# Security

## Что защищаем

- API keys
- OAuth tokens
- SSH keys
- cookies
- `.env`
- пользовательские сессии

## Основные правила

- Не коммитить секреты.
- Не запускать удалённые install-скрипты без проверки.
- Не включать MCP-серверы по популярности.
- Перед изменением пользовательских настроек делать backup.
