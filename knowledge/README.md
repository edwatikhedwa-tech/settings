# Knowledge Base

Здесь хранятся компактные, проверяемые выводы исследований. Исходные ролики, статьи и документы остаются ссылками; в Git сохраняются только собственные заметки, контекст и решения.

- `youtube/` - исследования по видеоматериалам.
- Для каждого исследования фиксируй выборку, ссылки, выводы, неизвестное и следующий шаг.
