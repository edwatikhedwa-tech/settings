# Личное окружение AI-агентов для разработки и исследований

- Status: active
- Started: 2026-07-19
- Last updated: 2026-07-19

## Goal

Настроить простое и эффективное взаимодействие пользователя с AI-агентами для разработки и исследований.

## Current State

Через `youtube_research_candidates` выполнены четыре русско-английских запроса: 55 уникальных результатов, 47 кандидатов после дедупликации и шесть детальных карточек. Добавлен бесплатный локальный мост `YouTube Research Capture`: он принимает текст, который пользователь уже скопировал из YouTube Summary, или открытую в YouTube расшифровку. Исходный текст хранится только в игнорируемом локальном inbox.

## Decisions

- Использовать `auto`-выборку с расширением до 50 кандидатов для популярных тем.
- Сохранять в Git только ссылки, короткие доказательства и выводы; не сохранять полные расшифровки.
- Стоимость и бесплатные лимиты внешних инструментов фиксировать в каждом отчёте.
- Не применять глобальные изменения конфигурации по материалам видео без отдельного подтверждения.
- Использовать YouTube Summary как разрешённый источник текста, если пользователь сам получил текст в расширении; не приравнивать отсутствие публичных субтитров к отсутствию текста в YouTube Summary.

## Next Action

Однократно загрузить локальное расширение `extensions/youtube-research-capture` в Chrome, затем для шести роликов нажать `Copy Transcript` в YouTube Summary и захватить текст. После этого обновить отчёт краткими проверяемыми выводами.
